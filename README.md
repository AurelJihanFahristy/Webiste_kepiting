# Website Penjualan Olahan Kepiting untuk UMKM

Sebuah proyek perangkat lunak berbasis web yang dirancang untuk membantu Usaha Mikro, Kecil, dan Menengah (UMKM) di bidang olahan kepiting untuk memasarkan produk mereka secara digital.

##  latar Belakang Masalah

Meskipun UMKM berperan penting dalam perekonomian, banyak pelaku usaha (khususnya produsen olahan kepiting) masih mengandalkan metode pemasaran konvensional. Hal ini mengakibatkan beberapa keterbatasan:

* Jangkauan pasar menjadi sempit.
* Upaya promosi kurang efektif.
* Proses transaksi dan pemesanan tidak efisien.

## 🎯 Solusi

Untuk mengatasi masalah tersebut, proyek ini menghadirkan solusi berupa website penjualan (e-commerce). Website ini berfungsi sebagai media promosi sekaligus sarana transaksi online untuk:

* Memperluas jangkauan pasar.
* Meningkatkan efisiensi transaksi.
* Memberikan pengalaman berbelanja yang praktis bagi konsumen.

## ✨ Fitur Utama

Website ini dirancang agar *user-friendly* (mudah digunakan) dan dilengkapi dengan fitur-fitur utama:

* **Galeri Produk:** Menampilkan daftar produk olahan kepiting secara menarik.
* **Halaman Admin:** Panel kontrol untuk mengelola produk, pesanan, dan konten website.
* **Dashboard:** Ringkasan aktivitas dan statistik penjualan (untuk admin).
* **Profil Perusahaan:** Halaman informasi mengenai UMKM terkait.

## 🛠️ Metode Pengembangan

Proyek ini dikembangkan menggunakan metode **Rapid Application Development (RAD)**. Metode ini dipilih karena menekankan kecepatan dan fleksibilitas, dengan tahapan yang melibatkan pengguna secara aktif:

1.  *Requirements Planning* (Perencanaan Kebutuhan)
2.  *User Design* (Perancangan Bersama Pengguna)
3.  *Construction* (Pembangunan Sistem)
4.  *Cutover* (Implementasi)
